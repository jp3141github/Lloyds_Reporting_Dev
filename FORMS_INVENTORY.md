## RRA Forms (Annual Return)

The Regulatory Return Annual (RRA) is submitted once per year and contains comprehensive syndicate data.

| Form | Name | Description | Status | In Codebase |
|------|------|-------------|--------|-------------|

| 020 | Exchange Rates | Currency rates used for translation to GBP | Active | ✅ V2 |

| 071 | SCOB Mapping | Lloyd's SCOB to Solvency II LOB mapping | Active | ✅ V2 |
| 081 | Reserving Class Definitions | Actuarial reserving class structure and methods | Active | ✅ V2 |
| 091 | LPT Data | Loss Portfolio Transfers and ADC transactions | Active | ✅ V2 |

| 291 | Claims Development Gross | Gross claims triangles by development year | Active | ✅ |

## Form Numbering Convention

Lloyd's forms follow a consistent numbering scheme:

| Range | Category |
|-------|----------|
| 0XX | General / Syndicate Information |
| 1XX | Premium and Income |
| 2XX | Claims and Reserves |
| 3XX | Actuarial and Loss Ratios |
| 4XX | Expenses |
| 5XX | Reinsurance |
| 6XX | Assets (rarely used at syndicate level) |
| 7XX-8XX | Reserved |
| 9XX | Supplementary / Sign-off |
